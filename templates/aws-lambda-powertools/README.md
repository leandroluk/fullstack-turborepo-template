# AWS Lambda Powertools Template

Este é um template básico para iniciar um projeto serverless utilizando o [AWS Lambda Powertools para TypeScript](https://docs.powertools.aws.dev/lambda/typescript/latest/).
Ele já está configurado com as ferramentas estáticas do monorepo (TypeScript, ESLint, Prettier, Vitest).

## Scripts disponíveis

- `npm run build`: Empacota o projeto utilizando esbuild
- `npm run lint`: Valida o projeto através do ESLint configurado no monorepo
- `npm run format`: Formata o código usando Prettier
- `npm run test`: Roda os testes com Vitest
