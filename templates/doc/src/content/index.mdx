# Welcome to Template

**Template** project using Next.js and already configured with monorepo configs

## Quick Start

Get your development environment up and running in seconds.

```bash
pnpm install
pnpm dev
```

## Project Structure

- **`template/`**: The main template environment.
  - **`api`**: The main Nest.js API environment.
  - **`app`**: The main Expo application environment.
  - **`doc`**: The main Nextra/Next.js documentation environment.
  - **`k8s`**: The main Kubernetes environment.
  - **`web`**: The main Next.js web environment.
- **`packages/`**: Shared packages.
  - **`config-eslint`**: Shared ESLint configuration.
  - **`config-typescript`**: Shared TypeScript configuration.
  - **`config-vitest`**: Shared Vitest configuration.
