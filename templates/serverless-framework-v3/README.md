# Serverless Framework v3 Template

Este é um template básico para iniciar um projeto utilizando o [Serverless Framework versão 3](https://www.serverless.com/).

## Scripts disponíveis

- `npm run build`: Empacota o projeto utilizando serverless e esbuild
- `npm run lint`: Valida o projeto através do ESLint configurado no monorepo
- `npm run format`: Formata o código usando Prettier
- `npm run test`: Roda os testes com Vitest
